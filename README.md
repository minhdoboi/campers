# Campers

VIBE coded game.

An isometric 2D prototype: a team of four campers on a
procedurally generated natural parcel — water, sand, grass, forest, and rock
built from a tile atlas. The land has up to 3 elevation levels connected by
dedicated directional ramp tiles, and each camper follows a recorded action
plan shown in the bottom HUD (select a person on the left to view their
sequence; double-click a name to center the camera on them).

Made with Godot 4.7.

## Run

Open the project in the Godot editor and press F5, or:

```sh
godot4 --path .
```

The splash screen starts the game. In-game controls: WASD/arrows or
middle-mouse drag to pan, mouse wheel to zoom, `R` to generate a new parcel,
`Esc` to return to the menu.

## Layout

- `scenes/splash/` — title screen (main scene)
- `scenes/game/` — game scene (3 elevation TileMapLayers + AStar navigation),
  camera, camper (action-queue NPC), and the bottom HUD panel
- `scripts/terrain_generator.gd` — noise-based island generation with 3
  quantized elevation levels, cliff smoothing, and ramp placement
- `resources/terrain_tileset.tres` — isometric TileSet (64x32 cells, 64x48
  block textures) over `assets/tiles/tiles.png`
- `assets/` — placeholder pixel art, generated by `tools/gen_assets.gd`

## Placeholder art

All art is generated. To tweak and regenerate it, edit `tools/gen_assets.gd` and run:

```sh
godot4 --headless --path . --script res://tools/gen_assets.gd
```

To replace it with real assets, drop new images over `assets/tiles/tiles.png`
(one row of 64x48 block tiles — 32px diamond top plus 16px cliff skirt — in
order: deep water, water, sand, grass, forest floor, rock),
`assets/tiles/ramps.png` (four 64x64 slope sprites rising toward +x, -x, -y,
+y), `assets/sprites/tree.png`, and `assets/sprites/camper.png`.

## Dev helper

Screenshot a scene without interacting:

```sh
godot4 --path . --script res://tools/screenshot_game.gd -- out.png [res://scene.tscn]
```
